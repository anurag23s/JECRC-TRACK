## Introduction

Welcome to JECRC Track: College Bus Management with Real-Time Bus Tracking and Route Information Portal. This solution is designed to revolutionize college bus transportation services by enhancing the overall commuting experience for students, drivers, and administrators.

JECRC Track is built on the core feature of Real-Time Bus Tracking, utilizing advanced GPS technology to display dynamic, live bus locations on a digital map. This innovative feature empowers drivers and students alike by providing precise, real-time bus location data, which helps optimize bus schedules, minimize waiting times, and enhance the overall efficiency of transportation services.

The system also includes a comprehensive Route Information Portal, a centralized platform that provides detailed information on bus routes, driver details, pickup and drop-off points, timings, and any schedule changes. Students can access this portal to plan their journeys effectively, ensuring they are well-informed about their routes and schedules.

The benefits of JECRC Track are significant, including improved efficiency through real-time tracking and route information, which allows students and drivers to optimize their commutes, reduce waiting times, and ensure timely arrivals and departures.



## AGENDA

The project aims to help students and administrators by improving the efficiency and safety of college bus transportation. The project provides a fully functional system in which students are informed about bus routes and schedules in real-time, enhancing their daily commute experience.

## Getting Started  

To get started with the project, follow these steps:

Clone the repository:
  	
   	- git clone https://github.com/yourusername/JECRCTrack.git
	- cd JECRCTrack

Install dependencies:

	- npm install

Run the application:

	- npm start

Build for Android/iOS:

	- npx react-native run-android
